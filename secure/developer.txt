3.  | Class                             | Purpose                             |
    | --------------------------------- | ----------------------------------- |
    | `text-lg`                         | Makes the text larger               |
    | `py-3 px-8`                       | Increases padding (bigger button)   |
    | `rounded-2xl`                     | More border radius for rounded look |
    | `hover:bg-white hover:text-black` | Clean hover                         |
    | `transition duration-300`         | Smooth effect                       |

4.  | Tailwind Class           | Font Size       | Example Use               |
    | ------------------------ | --------------- | ------------------------- |
    | `text-sm`                | 0.875rem (14px) | Small text or captions    |
    | `text-base`              | 1rem (16px)     | Default body text         |
    | `text-lg`                | 1.125rem (18px) | Slightly larger than base |
    | `text-xl`                | 1.25rem (20px)  | Headings or subtitles     |
    | `text-2xl`               | 1.5rem (24px)   | Subheadings               |
    | `text-3xl`               | 1.875rem (30px) | Prominent headings        |
    | `text-4xl`               | 2.25rem (36px)  | Hero section titles       |
    | `text-5xl`               | 3rem (48px)     | Large headings            |
    | `text-6xl` to `text-9xl` | Up to 128px     | Hero banners, branding    |
